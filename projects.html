Big Red Drop

Play Game

Project Overview

Big Red Drop is a WKU-themed Plinko-style game where players control falling balls through obstacles to score points. The game features physics-based gameplay with WKU's mascot Big Red and custom-designed assets.

Technologies Used

  • GDevelop 5 Game Engine
  • Adobe Illustrator (Asset Creation)
  • Event-Based Programming
Play Full Screen

Game Features

  • Physics-Based Gameplay: Realistic ball physics using GDevelop's physics engine
  • Score System: Dynamic point tracking with multipliers and high score saving
  • WKU Theme: Custom-designed assets featuring Big Red and WKU colors
  • Interactive Elements:
    • Responsive click-to-drop controls
    • Multiple drop zones and paths
    • Score multiplier zones
    • Obstacle interactions
  • Visual Feedback: Particle effects and animations for enhanced user experience
  • Mobile-Friendly: Touch controls for mobile devices

Development Process

  • Planning & Design:
    • Initial concept sketches and gameplay mechanics
    • WKU theme integration planning
    • Asset creation in Adobe Illustrator
  • Implementation:
    • GDevelop event system setup for game logic
    • Physics calibration for optimal gameplay
    • Score system development
    • UI/UX implementation
  • Testing & Refinement:
    • Playtesting for balance adjustments
    • Performance optimization
    • Bug fixes and polish

            document.addEventListener("DOMContentLoaded", function() {
                const wrapper = document.querySelector(".slideshow-wrapper");
                let currentIndex = 0;
            
                function updateSlideshow() {
                    const items = wrapper.children;
                    for (let i = 0; i < items.length; i++) {
                        if (i === currentIndex) {
                            items[i].style.display = "block";
                            if (items[i].querySelector("video")) {
                                items[i].querySelector("video").play();
                            }
                        } else {
                            items[i].style.display = "none";
                            if (items[i].querySelector("video")) {
                                items[i].querySelector("video").pause();
                            }
                        }
                    }
                }
            
                document.getElementById("prevBtn").addEventListener("click", () => {
                    currentIndex = (currentIndex - 1 + wrapper.children.length) % wrapper.children.length;
                    updateSlideshow();
                });
            
                document.getElementById("nextBtn").addEventListener("click", () => {
                    currentIndex = (currentIndex + 1) % wrapper.children.length;
                    updateSlideshow();
                });
            
                updateSlideshow();
            });
